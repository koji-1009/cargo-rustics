## Reporting issues

Before opening an issue:

- Reproduce against `main` from a clean checkout.
- Include the rustc version (`rustc --version`) and the cargo-rustics version (`cargo rustics --version`).
- Paste the relevant snippet from `cargo rustics analyze --reporter ai` if the issue is about a lens result.

## Pull requests

The PR template lives in `AGENTS.md`. Self-application gate is mandatory.
